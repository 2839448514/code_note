# 布尔字面量

### 语法

- `true`
- `false`

### 解释

布尔字面量是关键词 `true` 和 `false`。它们是 [[基础类型|bool]] 类型的[[值类别|纯右值]]。

### 注解

从 `bool` 到其他类型的隐式转换见[[隐式转换|整数转换]]，从其他类型到 `bool` 的隐式转换见[[隐式转换|布尔转换]]。

### 示例

运行此代码
```c++
#include <iostream>
int main()
{
  std::cout << std::boolalpha
            << true << '\n'
            << false << '\n'
            << std::noboolalpha
            << true << '\n'
            << false << '\n';
}
```

输出：

```c++
true
false
1
0
```
